## Keyword Research

Use WebSearch to research before writing. Process:

1. **Primary keyword**: One clear phrase the post targets. Place in title, first paragraph, slug, and meta_title.
2. **Search intent check**: Search the keyword. Note what Google features appear (featured snippet, people-also-ask, knowledge panel). Shape content to win those features.
3. **Long-tail variants**: Find 3-5 related queries. Weave them naturally into H2/H3 headings and body text.
4. **Competition analysis**: Check top 5 results for the keyword. Identify gaps in their coverage — those become your unique angles.

**Real estate keyword patterns that work:**
- "[Locality] property rates 2026" (informational)
- "best areas to buy in [city] under [budget]" (commercial)
- "[Project name] review / RERA status" (investigational)
- "[Policy] impact on [city] real estate" (news + analysis)
- "how to [action] in [city]" (guides)

## Title Optimization

Formula: `[Primary Keyword] — [Unique Hook] | [Location/Year]`

Rules:
- Max 60 characters for meta_title (can differ from display title)
- Display title can be longer but front-load the keyword
- Use numbers, brackets, or power words: "2026", "[Updated]", "Complete Guide", "Before You Buy"
- Never start with "How to" unless genuinely a guide (overused)

Examples:
- "Dwarka Expressway Property Rates Jump 18% — What Buyers Should Know (2026)"
- "Circle Rate Revisions in Gurgaon: Sectors Where Buying Just Got Cheaper"

## Meta Description

Rules:
- Max 155 characters
- Include primary keyword once
- End with a call-to-action or value proposition
- Do NOT repeat the title

Template: `"[Summary sentence with keyword]. [Why it matters for reader]. [Action/insight]."`

## Heading Structure

```
H1: Post Title (auto, do not add another H1)
H2: Primary Keyword / Main Section
  H3: Sub-section with long-tail keyword
  H3: Another sub-section
H2: Data / Analysis Section
  H3: Specific data point
H2: FAQ / Common Questions (wins People Also Ask)
  H3: Question 1?
  H3: Question 2?
H2: Sources (auto-generated by pipeline)
```

## URL Slug

- Use focus_keyword directly, hyphenated
- Max 5 words, no stop words
- Include year if time-sensitive: `dwarka-expressway-rates-2026`
- Never use articles: "the", "a", "an"

## Internal Linking

When writing, identify opportunities to link to:
- Other 360 Ghar blog posts (use slug in `seo_metadata.internal_links`)
- Property search pages: `https://360ghar.com/search?q=[locality]`
- Data Hub pages: circle rates, RERA project pages
- 360 Virtual Tour pages for specific projects

Aim for 3-5 internal links per post.

## Schema Markup

Include in `seo_metadata.schema_markup`. Choose based on content type:

**Article** (default): For news and analysis posts
```json
{
  "@context": "https://schema.org",
  "@type": "Article",
  "headline": "...",
  "datePublished": "...",
  "author": {"@type": "Organization", "name": "360 Ghar"}
}
```

**FAQPage**: When post has a Q&A section
```json
{
  "@context": "https://schema.org",
  "@type": "FAQPage",
  "mainEntity": [
    {"@type": "Question", "name": "...", "acceptedAnswer": {"@type": "Answer", "text": "..."}}
  ]
}
```

**HowTo**: For step-by-step guides
```json
{
  "@context": "https://schema.org",
  "@type": "HowTo",
  "name": "...",
  "step": [...]
}
```

## Tag & Category Strategy

**Categories** (broad, max 2 per post):
- Real Estate, Gurgaon, News, Policy, Guides, Market Analysis, Luxury, Affordable Housing, NRI, Commercial, Investment

**Tags** (specific, 5-8 per post):
- Include: locality name, topic, year, relevant project/developer names
- Match to popular search queries
- Use exact-match forms of secondary keywords

## Trending Score Guide

Set `seo_metadata.trending_score` (0-100):
- 80-100: Breaking news, viral topic, government policy just announced
- 60-79: Active trend, high search volume, multiple outlets covering
- 40-59: Steady interest, seasonal topic, evergreen content
- Below 40: Niche topic, low competition but also low volume
